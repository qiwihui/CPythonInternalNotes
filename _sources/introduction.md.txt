介绍
===

这本书将涵盖 CPython 内部实现的主要概念，并学习如何：

- 阅读和浏览源代码
- 从源代码编译 CPython
- 更改 Python 语法并将其编译到你的 CPython 版本中
- 导航并理解诸如列表、字典和生成器的概念的内部工作原理
- 掌握 CPython 的内存管理能力
- 使用并行和并发扩展你的 Python 代码
- 使用新功能修改核心类型
- 运行测试套件
- 分析和基准测试 Python 代码和运行时的性能
- 像专家一样调试 C 和 Python 代码
- 修改或升级 CPython 库的组件并将它们贡献给未来的版本
